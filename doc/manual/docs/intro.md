# EZPlayer Manual

EZPlayer sequences your lights and runs your show. This manual covers how the
system fits together and the interfaces it exposes.

- **[Architecture](./architecture/overview.md)** — how a show reaches your lights,
  from a high level down to the developer-facing internals.
- **[Reference](./reference/api.md)** — the API and other detailed specs.

> Diagrams in this manual are compiled from [d2](https://d2lang.com/) sources in
> `doc/assets/diagrams`, the same sources used by the slide decks.
